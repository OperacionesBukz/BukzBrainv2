- Use `Button` variant="ghost" size="icon" with className `"h-9 w-9 rounded-lg text-muted-foreground hover:text-foreground relative"`
- Render `<Bell className="h-4 w-4" />`
- If `unreadCount > 0`, render an absolute-positioned badge:
  ```tsx
  <span className="absolute -top-0.5 -right-0.5 flex h-4 min-w-4 items-center justify-center rounded-full bg-destructive px-1 text-[10px] font-bold text-destructive-foreground">
    {unreadCount > 99 ? "99+" : unreadCount}
  </span>
  ```
- Title attribute: `"Notificaciones"`

**PopoverContent:**
- `className="w-80 p-0"` with `align="end"` and `sideOffset={8}`
- Header div with:
  - `<h4 className="text-sm font-semibold">Notificaciones</h4>`
  - If `unreadCount > 0`: "Marcar todas como leidas" button (ghost, size sm) with `CheckCheck` icon
    - onClick: call `markAllNotificationsAsRead(user.email)` and close popover
- `<Separator />`
- If loading: show `<p className="p-4 text-sm text-muted-foreground text-center">Cargando...</p>`
- If notifications.length === 0: show `<p className="p-4 text-sm text-muted-foreground text-center">No tienes notificaciones</p>`
- Else: `<ScrollArea className="max-h-80">` containing notification items

**Each notification item:**
- A `button` element (full width, text-left) with:
  - `className={cn("flex flex-col gap-1 p-3 w-full text-left hover:bg-accent/50 transition-colors border-b border-border/50 last:border-0", !notification.read && "bg-accent/30")}`
  - onClick handler that:
    1. If not read, calls `markNotificationAsRead(notification.id)`
    2. Calls `navigate(notification.resourcePath)`
    3. Sets `open` to false
- Inside the button:
  - Row with: `<span className="text-sm font-medium">{notification.title}</span>` and if !read, a small blue dot: `<span className="h-2 w-2 rounded-full bg-primary shrink-0" />`
  - `<p className="text-xs text-muted-foreground line-clamp-2">{notification.message}</p>`
  - `<span className="text-[11px] text-muted-foreground/70">` with `formatDistanceToNow(notification.createdAt?.toDate() ?? new Date(), { addSuffix: true, locale: es })`

**Icon mapping** (optional, for visual variety):

  - `leave_request_approved`: text-green-500
  - `leave_request_rejected`: text-red-500
  - `leave_request_created`: text-yellow-500

All visible text MUST be in Spanish.
